##### About Image Animation:
* CSS style used: keyframes
->
* style iteration count is infinite, so the animation does not stop , the timing of one complete style is 1.5 seconds , the animation timing function is linear , therefore the style speed is uniform .
* transform :translate();
->
* shifts the image in x, y or both x-y direction .
* with the help of translate , the path of image rotation is octagon.
* transform: scale();->
* the scale after one complete rotation of image is set to 0.8 horizontally to squeeze the image and 1.2 vertically to stretch the image.
* Further , transition: ease-in-out, for smooth transition.
